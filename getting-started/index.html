Getting Started

This is a guide page to Start new project using vite bundler.

Start project using SASS without Tailwind

  1. Remove all tailwind related dependency
    2. npm uninstall tailwindcss autoprefixer postcss OR yarn remove tailwindcss autoprefixer postcss
  2. Remove all tailwind related files since we wont need them anymore.
    • tailwind.config.js
    • postcss.config.js
  3. Remove @tailwind base style in styles.scss 
    4. @tailwind base; -
@tailwind components; -
@tailwind utilities; -

    P.S. Some page including this page will be broken when Tailwind removed.

Start project using Tailwind with SASS

  1. 1. You are all set already, Good luck have fun!

Script Command

 npm install OR yarn install // to install dependency
 npm run dev OR yarn dev // to start dev environment with Hot Module Replacement
 npm run build OR yarn build // to build project
 npm run serve OR yarn serve // to serve builded project with Hot Module Replacement OFF

visit vite's official site guide to learn more about vite
